# AI/ML Portfolio

A curated collection of end‑to‑end machine learning, geospatial intelligence, and AI systems projects.  
Each project is structured as a standalone, production‑minded artifact.

## Portfolio Projects

### 1. Satellite Change Detection System Design
End‑to‑end ML system architecture for detecting changes in satellite imagery, including ingestion, preprocessing, model design, and deployment considerations.

### 2. ML Systems Scaling & Tradeoffs
Engineering analysis of scaling strategies, latency vs throughput, monitoring, and failure modes for real‑world ML systems.

### 3. RAG System Design & Evaluation
Retrieval‑augmented generation architecture, retrieval strategies, evaluation frameworks, and system‑level design.

### 4. AI Agent Architecture & Evaluation
Design and evaluation of autonomous and semi‑autonomous AI agents, including orchestration, memory, and tool‑use patterns.

### 5. GEOINT Change Detection & Object Detection
Hands‑on geospatial ML portfolio including chip generation, object detection, segmentation, and AOI‑specific workflows.

### 6. MLOps Deployment & Monitoring
Infrastructure‑as‑code, CI/CD, model deployment, monitoring, and drift detection for production ML systems.

### 7. Capstone: End‑to‑End ML System
A fully integrated ML system combining ingestion, training, deployment, monitoring, and evaluation.
