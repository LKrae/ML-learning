# 01 — ML System Design Fundamentals  
### Satellite Imagery Change Detection System (GEOINT)

This week focuses on developing a complete, senior‑level ML system design for a **Satellite Imagery Change Detection System**, aligned with GEOINT workflows and real‑world remote sensing constraints. The goal is to demonstrate end‑to‑end thinking across business framing, data engineering, modeling, deployment, monitoring, and lifecycle management.

This deliverable establishes a strong foundation for later phases of the 26‑week plan, including GEOINT modeling, RAG systems, agents, and MLOps.

---

## Objectives for Week 1
- Learn and apply ML system design fundamentals.
- Produce a complete, production‑grade system design document.
- Practice senior‑level reasoning across requirements, tradeoffs, and architecture.
- Build a reusable template for future system design exercises.
- Create portfolio‑ready artifacts for GitHub and interviews.
